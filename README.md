# 🩺 Melanoma Skin Cancer Detection using CNN

A deep learning project that classifies different types of skin cancer from dermoscopic images using a Convolutional Neural Network (CNN). The model is trained to recognize multiple categories of skin lesions and assist in the early detection of melanoma.

---

## 📌 Project Overview

Melanoma is one of the most dangerous forms of skin cancer. Early diagnosis can significantly improve treatment outcomes. This project uses a Convolutional Neural Network (CNN) built with TensorFlow and Keras to classify skin lesion images into multiple disease categories.

---

## ✨ Features

- Image classification using Deep Learning
- Custom CNN Architecture
- Data Augmentation

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/divya234-hue/Melanoma-Skin-Cancer-Detection.git
```

Move into the project directory

```bash
cd Melanoma-Skin-Cancer-Detection
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Environment

Windows

```bash
venv\Scripts\activate
```

Install Dependencies

```bash
pip install tensorflow keras numpy pandas matplotlib scikit-learn pillow opencv-python notebook
```
